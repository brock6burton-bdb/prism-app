# Prism Kitchen Backsplash Decision Engine

Working Next.js vertical slice with:
- 4-question homeowner flow
- DIY / Hybrid / Professional / Wait scoring
- Personalized strategy output
- Why alternatives ranked lower
- First-three-steps plan
- Know-when-to-stop conditions

## Run in GitHub Codespaces

```bash
npm install
npm run dev
```

Then open the forwarded port.

## Deploy

Import the repo into Vercel. It should detect Next.js automatically.

This is a prototype, not professional construction advice. It does not yet analyze photos, local codes, local pricing, hidden damage, or site-specific safety conditions.
